How to Use
1. Add students from Students section
2. Mark attendance every day per batch
3. SMS alerts send automatically at 3 absences
4. Add exam marks by selecting exam date
5. View results and batch reports anytime
6. Admin can reactivate inactive students
7. Attendance auto-archives every 35 days (keeps system fast)
Do's & Don'ts
✅ Mark attendance every class day
✅ Keep SMS balance topped up
✅ Only admin reactivates inactive students
✅ Contact developer for any issues
❌ Don't delete students — mark inactive instead
❌ Don't share admin password with anyone
❌ Don't edit Google Sheets directly
❌ Don't clear browser data
❌ Don't share login credentials
Auto Archive
📦 Attendance records older than 35 days are automatically moved to an "Archive_Attendance" sheet every 35 days at 3 AM.

✅ Keeps the system fast
✅ No data loss — everything stays in the same spreadsheet
✅ You can view archived data directly from Google Sheets
